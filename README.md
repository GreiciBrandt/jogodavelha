# Jogo da Velha

Um simples **Jogo da Velha** desenvolvido com **HTML, CSS e JavaScript puro**, onde dois jogadores se alternam entre **X** e **O** até que um vença ou ocorra empate.

## 📌 Funcionalidades

* Tabuleiro interativo 3x3
* Dois jogadores (X e O)
* Detecção automática de vitória
* Detecção de empate
* Botão para reiniciar o jogo
* Interface simples e fácil de usar

## 🛠️ Tecnologias utilizadas

* **HTML5** – Estrutura da página
* **CSS3** – Estilização do jogo
* **JavaScript (Vanilla JS)** – Lógica do jogo

## 🚀 Como executar o projeto

1. Clone este repositório:

```
git clone https://github.com/GreiciBrandt/jogodavelha.git
```

2. Acesse a pasta do projeto:

```
cd seu-repositorio
```

3. Abra o arquivo **index.html** no navegador.

Ou simplesmente dê **duplo clique no arquivo `index.html`**.
